# Smart Parking App

## Description
Smart Parking est une application mobile développée avec **Ionic + Angular** et **Capacitor**.  
Elle permet aux utilisateurs de réserver des places de parking et aux administrateurs de gérer les parkings de manière simple et rapide.

---

## Fonctionnalités principales
- **Authentification Firebase** avec deux rôles : Utilisateur et Administrateur
- **Réservation des places** pour les utilisateurs
- **Gestion des parkings** par les administrateurs (ajout, suppression)
- **Interface moderne** avec Ionic Tabs
- **Fonctionnalités natives** via Capacitor : GPS pour localisation des parkings, Camera pour scanner QR codes

---

## Acteurs
- **Utilisateur** :  
  - Créer un compte  
  - Réserver une place de parking  
  - Consulter l’historique de ses réservations

- **Administrateur** :  
  - Ajouter ou supprimer des parkings  
  - Voir toutes les réservations des utilisateurs

---

## Technologies utilisées
- **Frontend** : Ionic + Angular (Standalone Components)  
- **Backend / Auth** : Firebase Authentication + Firestore  
- **Fonctionnalités natives** : Capacitor (GPS, Camera)  
- **Versionning** : GitHub
